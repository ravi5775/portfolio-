# Pavan Innovate Hub - Portfolio

A modern, responsive portfolio website showcasing AI-powered projects and innovations.

## Features

- **Responsive Design**: Optimized for all devices
- **3D Animations**: Interactive 3D elements using Three.js
- **Project Showcase**: Highlighting AI, ML, and cloud projects
- **Modern UI**: Built with React, TypeScript, and Tailwind CSS
- **Performance Optimized**: Fast loading with Vite

## Technologies Used

- **Frontend**: React 18, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Icons**: Lucide React
- **3D Graphics**: Three.js, React Three Fiber

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd pavan-innovate-hub
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:8080](http://localhost:8080) in your browser.

### Build for Production

```bash
npm run build
```
